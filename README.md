# Invitations Management System

## Overview
This system manages the full invitation workflow for workshops and events, from sending emails to tracking responses.

Staff can send invitations to speakers and organizers, each with a unique link, and monitor responses in a centralized dashboard designed to be clear and easy to interpret.

## Features
- Send invitation emails to speakers and organizers
- Generate unique links for each recipient to track responses
- Track invitation status (sent, accepted, declined, no response)
- Dashboard view with clear, color-coded card-based layout for quick understanding
- Reduces reliance on manual tracking across email and spreadsheets

## Tech Stack
- PHP
- HTML
- CSS
- JavaScript
- SQL / MySQL
- WordPress (Formidable Forms integration)

## Key Contributions
- Built an end-to-end invitation workflow system used by staff for event coordination
- Implemented unique link tracking to reliably capture individual responses
- Designed a clear, card-based dashboard to make invitation status easy to understand at a glance
- Replaced fragmented tracking processes with a single, structured system

## Notes
This project focused on both backend workflow automation and front-end clarity, ensuring staff could not only collect responses but quickly understand the overall state of invitations without digging through data.
